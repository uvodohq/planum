## Overview

Planum provides a set of commonly used interface <b>icons</b> you can use in your project. The icons are exported as React components and can be used in your project as such or as a part of a custom component.

<br />

#### you need install @uvodohq/planum-icons as seperate package before use.

`npm install @uvodohq/planum-icons`

`import { UserIcon } from '@uvodohq/planum-icons'`
